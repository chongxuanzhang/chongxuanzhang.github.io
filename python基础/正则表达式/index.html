    正则表达式¶

    正则表达式是一个特殊的字符序列，它能帮助你方便的检查一个字符串是否与某种模式匹配。re 模块使 Python 语言拥有全部的正则表达式功能。compile 函数根据一个模式字符串和可选的标志参数生成一个正则表达式对象；该对象拥有一系列方法用于正则表达式匹配和替换。re 模块也提供了与这些方法功能完全一致的函数，这些函数使用一个模式字符串做为它们的第一个参数。

    1.1 语法简介¶

    由于正则表达式用于对字符串进行操作，因此我们将从最常见的任务开始：匹配字符。

    大多数字母和字符只会匹配自己。 例如，正则表达式 test 将完全匹配字符串 test 。

    但是，一些特殊的元字符或通配符除能表示它们的普通含义外；它们还可以匹配一些与众不同的东西，或者通过重复它们或改变它们的含义来影响正则的其他部分。

    • .

    默认匹配除换行符外的任意字符；如果指定了标签 DOTALL ，它将匹配包括换行符的任意字符。

    • [] 表示一个字符集合

    (1) 字符可以单独列出，例如：[amk]匹配a或m或k。

    (2) 通过-表示字符范围，例如：[a-z]只匹配小写字母；如果 - 进行了转义 （比如 [a\-z]）或者它的位置在首位或者末尾（如 [-a] 或 [a-]），它就只表示普通字符 '-'。

    (3) ^ 支持反取，例如：[^5] 将匹配除 '5' 之外的任何字符；如果插入符出现在字符类的其他位置，则它没有特殊含义。 例如：[5^] 将匹配 '5' 或 '^'。

    (4) 要匹配一个字符 ']'，可以通过转义符\：\[

    • ^

    (1) 匹配字符串开头

    (2) 在[]开始位置，表示反取

    • * 对它前面的正则式匹配0到任意次重复， 尽量多的匹配字符串。 ab* 会匹配 'a'，'ab'，或者 'a' 后面跟随任意个 'b'。

    • + 对它前面的正则式匹配1到任意次重复。 ab+ 会匹配 'a' 后面跟随1个以上到任意个 'b'，它不会匹配 'a'。

    • ?

    (1) 对它前面的正则式匹配0到1次重复。 ab? 会匹配 'a' 或者 'ab'。

    (2) 贪婪模式开关

    • {m} 对其之前的正则式指定匹配 m 个重复；少于 m 的话就会导致匹配失败。比如， a{6} 将匹配6个 'a' , 但是不能是5个。

    • {m,n} 对正则式进行 m 到 n 次匹配，在 m 和 n 之间取尽量多。 比如，a{3,5} 将匹配 3 到 5个 'a'。忽略 m 意为指定下界为0，忽略 n 指定上界为无限次。 比如 a{4,}b 将匹配 'aaaab' 或者1000个 'a' 尾随一个 'b'，但不能匹配 'aaab'。逗号不能省略，否则无法辨别修饰符应该忽略哪个边界。

    • {m,n}? 前一个修饰符的非贪婪模式，只匹配尽量少的字符次数。比如，对于 'aaaaaa'， a{3,5} 匹配 5个 'a' ，而 a{3,5}? 只匹配3个 'a'。

    • $ 匹配字符串的末尾。

    • \

    (1) 转义字符

    (2) 一些以 '\' 开头的特殊序列表示通常有用的预定义字符集，例如数字集、字母集或任何非空格的集合。

    模式 描述
    \d 匹配任何十进制数字；这等价于类 [0-9]。
    \D 匹配任何非数字字符；这等价于类 [^0-9]。
    \s 匹配任何空白字符；这等价于类 [ \t\n\r\f\v]。
    \S 匹配任何非空白字符；这相当于类 [^ \t\n\r\f\v]。
    \w 匹配任何字母、数字和_；这相当于类 [a-zA-Z0-9_]。
    \W 匹配任何非字母、数字和_；这相当于类 [^a-zA-Z0-9_]。
    • |

    A|B， A 和 B 可以是任意正则表达式，创建一个正则表达式，匹配 A 或者 B；任意个正则表达式可以用 '|' 连接。

    • (...) 分组

    • 正则表达式修饰符-可选参数：

    修饰符 描述
    re.I 使匹配对大小写不敏感
    re.L 做本地化识别（locale-aware）匹配
    re.M 多行匹配，影响 ^ 和 $
    re.S 使 . 匹配包括换行在内的所有字符
    re.U 根据Unicode字符集解析字符。这个标志影响 \w, \W, \b, \B
    re.X 该标志通过给予你更灵活的格式以便你将正则表达式写得更易于理解。
    • 匹配对象（Match Objects）属性和方法：

    一旦你有一个表示编译正则表达式的对象，你用它做什么？ 模式对象有几种方法和属性。 这里只介绍最重要的内容；请参阅 re 文档获取完整列表。

    匹配对象方法 描述
    group() 返回一个或者多个匹配的子组。不传参数或传入0，则输出整个匹配的字符串；如果它是一个范围 [1..99]，结果就是相应的括号组字符串；如果一个组号是负数，或者大于样式中定义的组数，一个 IndexError 索引错误就 raise。如果一个组包含在样式的一部分，并被匹配多次，就返回最后一个匹配。
    groups() 返回一个元组，包含所有匹配的子组，在样式中出现的从1到任意多的组合。 default 参数用于不参与匹配的情况，默认为 None。
    groupdict() 返回一个字典，包含了所有的 命名 子组。key就是组名。 default 参数用于不参与匹配的组合；默认为 None。
    start() 返回匹配的开始位置
    end() 返回匹配的结束位置
    span() 返回包含匹配 (start, end) 位置的元组

    1.2 处理函数介绍¶

    1.2.1 compile()¶

    re.compile(pattern,flags=0)将表达式编译成一个正则对象，通过对象match()、search()方法可以匹配对象。

    prog = re.compile(pattern)
result = prog.match(string)

    等价于

    result = re.match(pattern, string)

    如果需要多次使用这个正则表达式的话，使用 re.compile() 和保存这个正则对象以便复用，可以让程序更加高效。

    1.2.2 match()¶

    re.match(pattern, string, flags=0) 尝试从字符串的起始位置匹配一个模式，如果起始位置匹配不成功的话，match()就返回none。

    import re

# re.match()
string = '我是match函数'
s1 = re.match('我', string)
print(s1) # <re.Match object; span=(0, 1), match='我'>
print(s1.group()) # 我
print(s1.group(0)) # 我
print(s1.span()) # (0,1)

s2 = re.match('match',string)
print(s2) # None

s3 = re.match('(.*)match(.*)',string)
print(s3.group()) # 我是match函数
print(s3.group(0)) # 我是match函数
print(s3.group(1)) # 我是
print(s3.group(2)) # 函数
print(s3.groups()) # ('我是', '函数')
print(s3.groups()[0]) # 我是

    1.2.3 search()¶

    re.search(pattern, string, flags=0) 扫描整个字符串并返回第一个成功的匹配。

    import re

# re.search(pattern, string, flags=0)
string = '我是search函数'
s1 = re.search('search',string)
print(s1.group()) # search

    1.2.4 findall()¶

    re.findall(pattern, string, flags=0) 对 string 返回一个不重复的 pattern 的匹配列表， string 从左到右进行扫描，匹配按找到的顺序返回。如果样式里存在一到多个组，就返回一个组合列表；就是一个元组的列表（如果样式里有超过一个组合的话）;空匹配也会包含在结果里。

    import re

# re.findall(pattern, string, flags=0)
string = '我是findall函数，能匹配所有；match 和 search 是只匹配1次。'
s1 = re.findall('[a-z]+',string)
print(s1) # ['findall', 'match', 'search']

    1.2.5 finditer()¶

    re.finditer(pattern, string, flags=0)pattern 在 string 里所有的非重复匹配，返回为一个迭代器 iterator 保存了 匹配对象 。 string 从左到右扫描，匹配按顺序排列；空匹配也包含在结果里。

    import re

# re.finditer()
string = '我是findall函数，能匹配all；match 和 search 是只匹配one次。'
s1 = re.finditer('[a-z]+',string)
print(s1) # <callable_iterator object at 0x00000210AC2D6F70>
for s in s1:
    print(s.group())
# Output:
# findall
# all
# match
# search
# one

    1.2.6 split()¶

    re.split(pattern, string, maxsplit=0, flags=0)用 pattern 分开 string 。 如果在 pattern 中捕获到括号，那么所有的组里的文字也会包含在列表里。如果 maxsplit非零， 最多进行 maxsplit 次分隔， 剩下的字符全部返回到列表的最后一个元素。

    import re

# re.split()
string = '我是split函数，负责分割字符串。'
print(re.split('\W+', string)) # ['我是split函数', '负责分割字符串', '']
print(re.split('(\W+)', string)) # ['我是split函数', '，', '负责分割字符串', '。', '']
print(re.split('\W+', string,1)) # ['我是split函数', '负责分割字符串。']
print(re.split('a',string)) # ['我是split函数，负责分割字符串。'] 找不到，不分割

    1.2.7 sub()¶

    re.sub(pattern, repl, string, count=0, flags=0)

    import re
string = '我是subb函数，我能查找替换'
print(re.sub('subb', 'sub', string)) # 我是sub函数，我能查找替换
