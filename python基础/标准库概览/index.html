    标准库概览¶

    1.1 OS 模块¶

    os 模块提供了许多与操作系统交互的函数:

    # OS 模块
import os

print(os.name) # 获取操作系统 windows-nt
print(os.sep) # 路径分隔符：windos-'\'，其他-'/'
print(os.getcwd()) # 返回当前工作目录
os.chdir('路径')  # 更改当前工作目录
os.rmdir('路径') # 删除文件
os.rename('old','new') # 更改文件名
os.mkdir('123') # 创建文件夹123
print(os.listdir()) # 列出指定目录里的文件和文件夹，默认当前工作目录

# os.path
print(os.path.abspath('o1.py')) # 绝对路径；01.py为当前工作目录下的文件
print(os.path.isdir('test')) # True，判断是否是文件夹；test为当前工作目录下的文件
print(os.path.isfile('test')) # False，判断是否是文件
print(os.path.exists('test')) # True，判断是否存在
print(os.path.splitext('2020.2.21.demo.py')) # 分割
...

    一定要使用 import os 而不是 from os import * 。这将避免内建的 open() 函数被 os.open() 隐式替换掉，因为它们的使用方式大不相同。

    内置的 dir() 和 help() 函数可用作交互式辅助工具，用于处理大型模块，如 os:

    >>> import os
>>> dir(os)
<returns a list of all module functions>
>>> help(os)
<returns an extensive manual page created from the module's docstrings>

    1.2 sys 模块¶

    该模块提供了一些变量和函数；这些变量可能被解释器使用，也可能由解释器提供；这些函数会影响解释器。本模块总是可用的。

    # sys 模块
import sys

print(sys.argv) # 一个列表，其中包含了被传递给 Python 脚本的命令行参数。 
print(sys.path) # 列表：查找模块的路径；写的.py文件放在这个列表里，import时先在列表里的路径里寻找，没有就报错
sys.stdin # 接收用户收入，与input()相似
sys.stdout # 输出；可以改变默认输出位置
sys.stderr # 修改，可以改变默认'错误'输出位置
sys.exit() # 退出程序，与内置函数exit()功能一样；内部可传入退出码（默认为0）；此代码后的代码不执行

    argv[0]为脚本的名称（是否是完整的路径名取决于操作系统）。如果是通过 Python 解释器的命令行参数 -c 来执行的， argv[0]会被设置成字符串 '-c' 。如果没有脚本名被传递给 Python 解释器， argv[0]为空字符串。

    1.3 math 模块¶

    math 模块提供对浮点数学的底层C库函数的访问:

    # math 模块
import math

print(math.ceil(5 / 2)) # 向上取整
print(5//2) # 向下取整
print(math.floor(5//2)) # 向下取整
print(math.fabs(-3)) # 绝对值
print(math.factorial(4)) # 求阶乘
print(math.fmod(5, 2)) # 余数，使用浮点数时首选
print(5 % 2) # 余数，使用整数时首选
print(math.modf(3.14)) # 返回小数和整数部分
print(math.pi) # pai

    1.4 random 模块¶

    random 模块提供了进行随机选择的工具:

    # random 模块
import random

print(random.random()) # 生成[0,1)的随机浮点数
print(random.randint(2,9)) # 生成[2,9]的随机整数
print(random.randrange(2, 9)) # 生成[2,9)的随机整数，默认步长为1

# random.choice() 从一个非空序列中随机抽取一个数据，如果序列为空，返回 IndexError
print(random.choice(['zhangsan','lisi','wangwu']))

# random.choices() 随机生成k大小的数据，如果不指定weights或cum_weights参数，则使用同等概率；数据可重复出现
print(random.choices(['zhangsan','lisi','wangwu'],k=3))

# random.sample() 从序列或集合中随机生成k大小的数据，数据不能重复
print(random.sample(['zhangsan','lisi','wangwu'],k=3))

# random.shuffle() 随机打乱位置
a = list('abcdefgh')
print(f'原始数据是：{a}') # 原始数据是：['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']
random.shuffle(a)
print(f'打乱顺序后的数据是：{a}') # 打乱顺序后的数据是：['h', 'e', 'g', 'd', 'b', 'a', 'c', 'f']；每次运行结果不同

    1.5 datetime 模块¶

    datetime 模块提供用于处理日期和时间的类；包括：date类、datetime(date)类、IsoCalendarDate(tuple)类、time类、timedelta类、tzinfo类、timezone(tzinfo)类。

    这些类型的对象都是不可变的。

    子类关系

    object
    timedelta
    tzinfo
        timezone
    time
    date
        datetime

    以下只是简单介绍date类的用法，更多类用法介绍请查阅官方教程，或查看相应内置函数介绍。

    # date 类
import datetime as dt # 导入，是为了介绍类的调用方法
from datetime import date

now = date.today() # 或者 dt.date.today() 用内置类的调用方法
print(f'当前日期是：{now}') # 当前日期是 ：2021-04-24

# "Return ctime() style string."
print(now.ctime()) # Sat Apr 24 00:00:00 2021

print(now.day) # 返回日 24
print(now.month) # 返回月 4
print(now.year) # 返回年 2021

# now.weekday() "Return day of the week, where Monday == 0 ... Sunday == 6."
print(now.weekday()) # 2021-04-24是周六，返回值为5
print(now.isoweekday()) # 返回周六
print(now.strftime('%d/%m/%y')) # 格式化日期: 24/04/21

# replace() """Return a new date with new values for the specified fields."""
print(now.replace(year=1990)) # month、day不改变，就使用之前的值

birthday = date(1999,8,23)
print(birthday) # 1999-08-23

# 综合用法
print(f'今天是{now}，周{now.isoweekday()}，距离你的出生{birthday}，已过了{(now-birthday).days}天了。')
# 今天是2021-04-24，周6，距离你的出生1999-08-23，已过了7915天了。

    glob 模块提供了一个在目录中使用通配符搜索创建文件列表的函数:

    # glob 模块
import glob

print(glob.glob('*.py')) # 列出当前目录下所有的.py文件

    1.7 re 模块¶

    re 模块为高级字符串处理提供正则表达式工具。对于复杂的匹配和操作，正则表达式提供简洁，优化的解决方案；后续会有详细章节来记录学习。

    # re 模块
import re

print(re.findall(r'\bf[a-z]*', 'which foot or hand fell fastest')) # ['foot', 'fell', 'fastest']
print(re.sub(r'(\b[a-z]+) \1', r'\1', 'cat in the the hat')) # cat in the hat
print('tea for too'.replace('too', 'two')) # tea for two

    1.8 互联网访问¶

    有许多模块可用于访问互联网和处理互联网协议。其中两个最简单的 urllib.request 用于从URL检索数据，以及 smtplib 用于发送邮件；后续会有详细章节来记录学习。

    >>> from urllib.request import urlopen
>>> with urlopen('http://tycho.usno.navy.mil/cgi-bin/timer.pl') as response:
...     for line in response:
...         line = line.decode('utf-8')  # Decoding the binary data to text.
...         if 'EST' in line or 'EDT' in line:  # look for Eastern Time
...             print(line)

<BR>Nov. 25, 09:43:32 PM EST

>>> import smtplib
>>> server = smtplib.SMTP('localhost')
>>> server.sendmail('soothsayer@example.org', 'jcaesar@example.org',
... """To: jcaesar@example.org
... From: soothsayer@example.org
...
... Beware the Ides of March.
... """)
>>> server.quit()

    （请注意，第二个示例需要在localhost上运行的邮件服务器。）

    1.9 数据压缩¶

    常见的数据存档和压缩格式由模块直接支持，包括：zlib, gzip, bz2, lzma, zipfile 和 tarfile。

    >>> import zlib
>>> s = b'witch which has which witches wrist watch'
>>> len(s)
41
>>> t = zlib.compress(s)
>>> len(t)
37
>>> zlib.decompress(t)
b'witch which has which witches wrist watch'
>>> zlib.crc32(s)
226805979

    1.10 性能测量¶

    一些Python用户对了解同一问题的不同方法的相对性能产生了浓厚的兴趣。 Python提供了一种可以立即回答这些问题的测量工具。

    例如，元组封包和拆包功能相比传统的交换参数可能更具吸引力。timeit 模块可以快速演示在运行效率方面一定的优势:

    >>> from timeit import Timer
>>> Timer('t=a; a=b; b=t', 'a=1; b=2').timeit()
0.57535828626024577
>>> Timer('a,b = b,a', 'a=1; b=2').timeit()
0.54962537085770791

    与 timeit 的精细粒度级别相反， profile 和 pstats 模块提供了用于在较大的代码块中识别时间关键部分的工具。

    1.11 测试模块¶

    开发高质量软件的一种方法是在开发过程中为每个函数编写测试，并在开发过程中经常运行这些测试。

    doctest 模块提供了一个工具，用于扫描模块并验证程序文档字符串中嵌入的测试。测试构造就像将典型调用及其结果剪切并粘贴到文档字符串一样简单。这通过向用户提供示例来改进文档，并且它允许doctest模块确保代码保持对文档的真实:

    def average(values):
    """Computes the arithmetic mean of a list of numbers.

    >>> print(average([20, 30, 70]))
    40.0
    """
    return sum(values) / len(values)

import doctest
doctest.testmod()   # automatically validate the embedded tests

    unittest 模块不像 doctest 模块那样易于使用，但它允许在一个单独的文件中维护更全面的测试集:

    import unittest

class TestStatisticalFunctions(unittest.TestCase):

    def test_average(self):
        self.assertEqual(average([20, 30, 70]), 40.0)
        self.assertEqual(round(average([1, 5, 7]), 1), 4.3)
        with self.assertRaises(ZeroDivisionError):
            average([])
        with self.assertRaises(TypeError):
            average(20, 30, 70)

unittest.main()  # Calling from the command line invokes all tests
