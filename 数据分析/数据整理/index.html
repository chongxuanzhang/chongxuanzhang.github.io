    数据分组¶

    pandas对象可以在任何轴上拆分,分组的抽象定义是提供标签到分组名称的映射。

    import pandas as pd
import numpy as np
df = pd.DataFrame(
    [("bird", "Falconiformes", 389.0),
     ("bird", "Psittaciformes", 24.0),
     ("mammal", "Carnivora", 80.2),
     ("mammal", "Primates", np.nan),
     ("mammal", "Carnivora", 58),],
    index=["falcon", "parrot", "lion", "monkey", "leopard"],
    columns=("class", "order", "max_speed"),)
df
    class order max_speed
    falcon bird Falconiformes 389.0
    parrot bird Psittaciformes 24.0
    lion mammal Carnivora 80.2
    monkey mammal Primates NaN
    leopard mammal Carnivora 58.0 
    grouped = df.groupby("class") # 默认 axis = 0 ; df.groupby('A')只是语法糖df.groupby(df['A'])
grouped = df.groupby("order", axis="columns")# axis = 1 按列
grouped = df.groupby(["class", "order"]) # 按多列分组

    排序¶

    在groupby()过程中是默认对数据进行排序的，可以通过设置参数sort = False来取消自动排序。

    df2 = pd.DataFrame({"X": ["B", "B", "A", "A"], "Y": [1, 2, 3, 4]})
df2.groupby(["X"]).sum()
    Y
    X
    A 7
    B 3 
    df2.groupby(["X"], sort=False).sum()
    Y
    X
    B 3
    A 7

    缺失值¶

    在groupby()过程中是默认对NA进行删除的，可以通过设置参数dropna = False来取消。

    df_list = [[1, 2, 3], [1, None, 4], [2, 1, 3], [1, 2, 2]]
df_dropna = pd.DataFrame(df_list, columns=["a", "b", "c"])
df_dropna
    a b c
    0 1 2.0 3
    1 1 NaN 4
    2 2 1.0 3
    3 1 2.0 2 
    df_dropna.groupby(by=["b"], dropna=True).sum() # 默认 dropna = True
    a c
    b
    1.0 2 3
    2.0 2 5 
    df_dropna.groupby(by=["b"], dropna=False).sum()
    a c
    b
    1.0 2 3
    2.0 2 5
    NaN 1 4

    groupby对象属性¶

    groups是个字典。The groups attribute is a dict whose keys are the computed unique groups and corresponding values being the axis labels belonging to each group.

    df
    class order max_speed
    falcon bird Falconiformes 389.0
    parrot bird Psittaciformes 24.0
    lion mammal Carnivora 80.2
    monkey mammal Primates NaN
    leopard mammal Carnivora 58.0 
    df.groupby('class').groups

    {'bird': ['falcon', 'parrot'], 'mammal': ['lion', 'monkey', 'leopard']}

    可以遍历各组数据

    grouped = df.groupby("class")
for name,group in grouped:
    print(name)
    print(group)

    数据聚合¶

    创建groupBy对象后，可以使用多种方法对分组数据进行计算,aggregate()或等效 agg()是一种方法。

    df.groupby('class').agg(sum)
    max_speed
    class
    bird 413.0
    mammal 138.2 
    df.groupby(['class','order']).sum()
    max_speed
    class order
    bird Falconiformes 389.0
    Psittaciformes 24.0
    mammal Carnivora 138.2
    Primates 0.0

    运用多个函数

    df.groupby('class')['max_speed'].agg([np.sum,np.mean,np.std])
    sum mean std
    class
    bird 413.0 206.5 258.093975
    mammal 138.2 69.1 15.697771

    命名聚合

    animals = pd.DataFrame(
   {
        "kind": ["cat", "dog", "cat", "dog"],
        "height": [9.1, 6.0, 9.5, 34.0],
         "weight": [7.9, 7.5, 9.9, 198.0],
    } )
animals
    kind height weight
    0 cat 9.1 7.9
    1 dog 6.0 7.5
    2 cat 9.5 9.9
    3 dog 34.0 198.0 
    animals.groupby("kind").agg(
    min_height=("height", "min"),
    max_height=("height", "max"),
    average_weight=("weight", np.mean),)
    min_height max_height average_weight
    kind
    cat 9.1 9.5 8.90
    dog 6.0 34.0 102.75 
    animals.groupby("kind").agg(
    min_height=pd.NamedAgg(column="height", aggfunc="min"),
    max_height=pd.NamedAgg(column="height", aggfunc="max"),
    average_weight=pd.NamedAgg(column="weight", aggfunc=np.mean),)
    min_height max_height average_weight
    kind
    cat 9.1 9.5 8.90
    dog 6.0 34.0 102.75 
    animals.groupby("kind").agg(
     **{
         "total weight": pd.NamedAgg(column="weight", aggfunc=sum)}
)
    total weight
    kind
    cat 17.8
    dog 205.5 
    # 对序列同样有效
animals.groupby("kind").height.agg(
    min_height="min",
    max_height="max", )
    min_height max_height
    kind
    cat 9.1 9.5
    dog 6.0 34.0

    将不同函数运用到不同列

    animals.groupby('kind').agg({'height':'min','weight':'max'}) # 只传入函数名，不带括号；也可以传入lambda函数
    height weight
    kind
    cat 9.1 9.9
    dog 6.0 198.0

    apply()¶

    对分组数据进行的某些操作可能既不适合聚合类别，也不适合转换类别；或者，您可能只是希望groupBy推断如何合并结果。对于这些，可以使用apply的功能；apply可以处理一些特殊的用例。

    data = pd.DataFrame(
     {
         "A": ["foo", "bar", "foo", "bar", "foo", "bar", "foo", "foo"],
         "B": ["one", "one", "two", "three", "two", "two", "one", "three"],
         "C": np.random.randn(8),
         "D": np.random.randn(8),
     } )
