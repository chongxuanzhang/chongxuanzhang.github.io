    Scrapy基础用法¶

    利用Scrapy爬取官网推荐的http://quotes.toscrape.com/网站，来展示Scrapy的基本使用方法。解析数据使用的是BeautifuSoup，后续将进一步学习Scrapy的自带解析器怎么用？如何带参数请求数据？如何写入cookies？如何发送邮件？如何与selenium联动？如何完成超厉害的分布式爬虫……

    1. 虚拟环境创建¶

    官方文档推荐在虚拟模块里安装scrapy模块；在虚拟环境里可以使用pip安装需要的模块版本，并确保与已安装的相应系统软件包不冲突。

    用于创建和管理虚拟环境的模块称为 venv。venv 通常会安装你可用的最新版本的 Python。如果您的系统上有多个版本的 Python，可以通过运行 python3 或想要的任何版本来选择特定的Python版本。

    创建虚拟环境：1. win+R输入cmd进入终端；2. 切换到要保存虚拟环境的目录；3. 

    python3 -m venv myfirst-env

    如果 myfirst-env 目录不存在，它将为你创建一个，并在其中创建包含Python解释器，标准库和各种支持文件的副本的目录。4. 激活虚拟环境，切换路径到activate.bat所在目录，

    myfirst-env\Scripts\activate.bat

    激活虚拟环境将改变你所用终端的提示符，以显示你正在使用的虚拟环境，并修改环境以使 python 命令所运行的将是已安装的特定 Python 版本。

    可以使用 pip 来安装、升级和移除软件包；例如：

    # 安装requests的指定版本，python -m 是为获得权限
python -m pip install requests==2.6.0 
# 升级
python -m pip install --upgrade requests
# 删除 pip uninstall 后跟一个或多个包名称将从虚拟环境中删除包。
# 显示有关特定包的信息
pip show requests
# 显示虚拟环境中安装的所有软件包
pip list

pip freeze > requirements.txt # 将已安装的包放入文件中
python -m pip install -r requirements.txt # 安装文件里的包

    输入deactivate来结束虚拟环境。

    2. 创建Scrapy项目¶

    scrapy startproject tutorial [project_dir]

    这将在project_dir下创建一个tutorial项目（如果不设置路径，则与tutorial一致），包含以下内容：

    tutorial/
    scrapy.cfg            # deploy configuration file

    tutorial/         # project's Python module, you'll import your code from here
        __init__.py

        items.py          # project items definition file 用来定义数据

        middlewares.py    # project middlewares file

        pipelines.py      # project pipelines file 用来处理数据

        settings.py       # project settings file 各种设置

        spiders/          # a directory where you'll later put your spiders
            __init__.py

    2.1 定义数据¶

    在items.py文件里定义数据，这里要爬取引用的话和作者，所以定义数据如下：

    # Define here the models for your scraped items
#
# See documentation in:
# https://docs.scrapy.org/en/latest/topics/items.html

import scrapy

class TutorialItem(scrapy.Item):
    # define the fields for your item here like:
    # name = scrapy.Field()
    author = scrapy.Field()
    content = scrapy.Field()

    2.2 编写爬虫程序¶

    在spiders文件夹下创建一个quotes_spider.py的文件来编写爬虫，文件内容如下：

    # 导入所需模块
import scrapy
from bs4 import BeautifulSoup
# 需要引用QutotesItem，它在items里面。因为是items在quotes_spider.py的上一级目录，所以要用..items，这是一个固定用法。
from ..items import QutotesItem

class QuotesSpider(scrapy.Spider):
    ''' 定义一个继承了scrapy.Spider的类QuotesSpider '''

    # 定义爬虫的名字为quotes
    name = 'quotes'
    # 定义爬取网站网址的域名，排除其他广告等的链接
    allowed_domains =['quotes.toscrape.com']
    # 定义起始网址，网站共有10页
    start_urls = []
    for page in range(1,11):
        url = f'http://quotes.toscrape.com/page/{page}/'
        start_urls.append(url)


    def parse(self,response):
        ''' parse是默认处理response的方法，不需要自己调用requests.get()'''

        soup = BeautifulSoup(response.text,'html.parser')
        datas = soup.find_all('div',class_='quote')
        for data in datas:
            # 实例化QutotesItem类
            item = QutotesItem()
            item['author'] = data.find('small',class_='author').text
            item['content'] = data.find('span',class_='text').text
            # 将获得的item传递给引擎
            yield item

    2.3 设置¶

    在settings.py文件里可以定义用户协议，是否遵守爬虫协议等；这里只修改用户协议和是否遵守爬虫协议及延迟时间：

    # Crawl responsibly by identifying yourself (and your website) on the user-agent
from fake_useragent import UserAgent
USER_AGENT = UserAgent().chrome

# Obey robots.txt rules
ROBOTSTXT_OBEY = True
DOWNLOAD_DELAY = 1

    2.4 写入CSV¶

    存储成csv文件的方法比较简单，只需在settings.py文件里，添加如下的代码即可：

    FEED_URI='./quotes_data.csv'  # 导出文件的路径
FEED_FORMAT='csv'  # 导出文件格式
FEED_EXPORT_ENCODING='ansi' # 导出文件编码；ansi是一种在windows上的编码格式

    2.5 写入Excel¶

    存储成Excel文件的方法要稍微复杂一些，我们需要先在settings.py里设置启用ITEM_PIPELINES，设置方法如下：

    # Configure item pipelines
# See https://docs.scrapy.org/en/latest/topics/item-pipeline.html
#ITEM_PIPELINES = {
#    'tutorial.pipelines.TutorialPipeline': 300,
#}

    只要取消ITEM_PIPELINES的注释（删掉#）即可。

    #取消`ITEM_PIPELINES`的注释后：

# Configure item pipelines
# See https://docs.scrapy.org/en/latest/topics/item-pipeline.html
ITEM_PIPELINES = {
    'tutorial.pipelines.TutorialPipeline': 300,
}

    可以去编辑pipelines.py文件：

    # Define your item pipelines here
#
# Don't forget to add your pipeline to the ITEM_PIPELINES setting
# See: https://docs.scrapy.org/en/latest/topics/item-pipeline.html


# useful for handling different item types with a single interface
from itemadapter import ItemAdapter
import openpyxl

class TutorialPipeline:

    def __init__(self):
        ''' 初始化函数 当类实例化时这个方法会自启动 '''

        self.wb = openpyxl.Workbook()
        self.ws = self.wb.active
        self.ws.append(['作者','名句'])

    def process_item(self, item, spider):
        ''' process_item是默认的处理item的方法，就像parse是默认处理response的方法 '''

        line = [item['author'],item['content']]
        self.ws.append(line)
        # 将item丢回给引擎，如果后面还有这个item需要经过的itempipeline，引擎会自己调度
        return item

    def close_spider(self,spider):
        ''' close_spider是当爬虫结束运行时，这个方法就会执行 '''

        self.wb.save('./quotes_data.xlsx')
        self.wb.close()

    2.6 运行项目¶

    1. 本地终端，切换到项目路径：scarpy crawl quotes

    2. 在与scrapy.cfg同级目录里创建一个main.py文件，内容如下：

    #导入cmdline模块,可以实现控制终端命令行。
from scrapy import cmdline

#用execute（）方法，输入运行scrapy的命令
cmdline.execute(['scrapy','crawl','quotes'])
